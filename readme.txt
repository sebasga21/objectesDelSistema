Hoja de trabajo "M06_03_FT_01 - Objectes del Sistema"

	- El primer y único ejercicio es una imitación del juego Flappy Bird pero mucho más sencillo. El objetivo es mostrar el método 
toFixed() cuando en cada iteración de la animación que le hemos puesto al agujero se le cambia la propiedad top, porque el número random que 
nos sale de Math.random() lo queremos sin decimales. También queremos mostrar la fecha para eso creamos un objeto Date y luego recogemos 
el dia, el mes y el año. Para mostrar la fecha, con el resultado y con un mensaje de "Game over" usamos el método concat().
